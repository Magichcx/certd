# 快速开始
本章节介绍如何快速开始使用`Certd`

## 一、 demo在线体验

官方DEMO地址，自助注册后体验

https://certd.handsfree.work/

> 注意数据将不定期清理，不定期停止定时任务，生产使用请自行部署
